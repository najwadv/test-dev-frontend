# Technical Test - Frontend Developer

**Nama:** Najwa Diva Azzahra
**Tanggal Pengerjaan:** 6 - 9 Februari 2026

Project ini berisi hasil pengerjaan technical test untuk posisi Frontend pada PT. Integrasia Utama.

### Project Description

Aplikasi ini bertujuan untuk menampilkan hasil dari instruksi-instruksi yang diberikan pada technical test Software Development oleh PT. Integrasia Utama untuk posisi Frontend. Aplikasi ini menggunakan React JS dan Tailwind CSS untuk menampilkan output dari program dan masih berjalan di local.

### Tech Stack

- React JS
- Tailwind CSS
- Vite
- React Hooks

### Cara Install dan Run Project

1. **Clone Repository**
   git clone https://github.com/najwadv/test-dev-frontend.git
   cd test-dev-frontend

2. **Install Dependencies**
   npm install

3. **Run Project**
   npm run dev

**Aplikasi akan berjalan di localhost:5173**

### Penggunaan AI Assistant

Dalam proses pembuatan program ini, kandidat menggunakan AI Assistant Gemini AI Pro. Berikut prompt yang digunakan dalam proses pembuatan:

"How to make the button and card into reusable components?"
"Can you explain the instructions in details? Give me references to solve it, but don't give me the code."
"I want to make a function to check if two words are anagram or not. I want it to have a proper logic and count the string length. Make it ignore the capitals and spaces."
"I have JSON data consists of list of stuffs (name, category, code, total). I need the data to be grouped like category > code > items. I have to add the total on each levels."
"Can you show me a proper data structure in React JS?"
"Based on the instruction, where should I show the output? Should I show it on the local/UI? Or let the function create a new file under src/assets/json?"
